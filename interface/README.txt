Interface graphique
====================

Détail des fichiers :
---------------------

- ressources.qrc et images : pour utiliser dans images dans les QButtons.
- mainwindow : la fenêtre principale.
- main : le lancement de la fenêtre (très court).

L'aspect général de la fenêtre a été créé à l'aide de Qt Designer.

Compilation :
-------------

On peut voir un aperçu de la fenêtre directement avec Qt Creator (touche Run, on Ctrl+R)

Pour compiler en ligne de commande, entrer ces deux commandes :

$ qmake
$ make
